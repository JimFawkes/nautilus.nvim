## Setup
Set the mode either to `blue` (default) or `grey`
```
  use {'pwntester/nautilus.nvim', config = function()
    require'nautilus'.setup({mode = "grey"})
  end}
```
## Requirements
Install `https://github.com/rktjmp/lush.nvim`
